mathpunch Music Player (BETA)

Here is a simple little music player I made for my personal website. It is small, lightweight, and mobile friendly! Just look at the bottom (bottom right if you are on bigger screens) and click on the play button.

It is also open source! You can find the source code here.

More details about how this all works and how you can implement it in your site is on the GitHub README so go check that out if you want to use this in your own projects
